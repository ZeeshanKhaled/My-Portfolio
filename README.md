# My Portfolio

This is my personal portfolio website, created to showcase my work, skills, CV, and projects in one place. The site presents my background as a Computer Science and AI student, alongside projects involving web development, game-inspired interfaces, UI/UX design, and creative digital ideas.

## Live Website

Access the project via this link: [My Portfolio](https://zeeshankhaled.github.io/My-Portfolio/)

## About the Website

The portfolio includes:

- A homepage introducing who I am and what I do
- An About page explaining my interests, approach, and future direction
- A Portfolio page showcasing my projects
- A CV page with a downloadable CV
- Responsive styling for different screen sizes
- A dark, tech-inspired visual design

## Featured Project Areas

Some of the project types shown in this portfolio include:

- Web development projects
- Browser-based games
- UI/UX concepts
- University coursework
- AI-inspired product ideas

The purpose of this website is to act as a central place where visitors can learn more about me, view my work, and access my CV.
